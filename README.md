# 🚀 Research & Content Generation using CrewAI

## 📝 Overview
This project leverages **CrewAI** to automate research and content writing on the topic of **"Medical Industry using Generative AI."** It consists of two intelligent agents:

1. 🧠 **Senior Research Analyst** - Gathers and summarizes the latest information.  
2. ✍️ **Content Writer** - Transforms research insights into engaging blog content.

## 🌟 Features
✅ Automated research on medical industry advancements using Generative AI.  
✅ SEO-optimized content creation with structured formatting.  
✅ Ensures factual accuracy and proper citation of sources.

## ⚙️ Installation

### 1️⃣ Clone this repository:
```bash
git clone https://github.com/your-repo-name.git
cd your-repo-name
```

### 2️⃣ Install dependencies:
```bash
pip install -r requirements.txt
```

### 3️⃣ Set up environment variables:
```bash
cp .env.example .env
```
🛠️ Fill in the required API keys in **`.env`**.

---

## 🚀 Usage
Run the script using:
```bash
python main.py
```
🎯 The agents will research and generate an insightful blog post on the given topic.
